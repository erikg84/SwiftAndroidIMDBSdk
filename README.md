# SwiftAndroidSDK

A cross-platform Swift SDK — business logic and utilities shared between **iOS** and **Android**.

## Requirements

| Platform | Minimum version |
|----------|----------------|
| iOS      | 15.0           |
| macOS    | 12.0           |
| Android  | API 28 (Android 9) |

---

## iOS Integration (Swift Package Manager)

### Xcode
1. **File → Add Package Dependencies**
2. Enter the repository URL: `https://github.com/YOUR_ORG/SwiftAndroidSdk`
3. Choose version rule: **Up to Next Major** from `1.0.0`

### Package.swift
```swift
dependencies: [
    .package(url: "https://github.com/YOUR_ORG/SwiftAndroidSdk", from: "1.0.0"),
],
targets: [
    .target(
        name: "YourApp",
        dependencies: [
            .product(name: "SwiftAndroidSDK", package: "SwiftAndroidSdk")
        ]
    ),
]
```

### Quick start (Swift)
```swift
import SwiftAndroidSDK

let sdk = SwiftAndroidSDK()
print(sdk.sdkVersion())           // "1.0.0"
print(reverseString("hello"))     // "olleh"
print(base64Encode("Swift!"))     // "U3dpZnQh"
print(isValidJSON(#"{"ok":true}"#)) // true
```

## Android Integration (Gradle)

### build.gradle
```groovy
dependencies {
    implementation 'io.github.YOUR_ORG:swift-android-sdk:1.0.0'
}
```

**minSdkVersion must be ≥ 28.**

### Quick start (Kotlin)
```kotlin
import io.github.YOUR_ORG.swiftandroidsdk.SwiftAndroidSDK
import io.github.YOUR_ORG.swiftandroidsdk.SwiftAndroidSDKKt

val sdk = SwiftAndroidSDK()
println(sdk.sdkVersion())                    // "1.0.0"
println(SwiftAndroidSDKKt.reverseString("hello")) // "olleh"
println(SwiftAndroidSDKKt.base64Encode("Swift!")) // "U3dpZnQh"
```

> **Note:** Java wrapper class names are generated by `swift-java`. The exact package/class names above may vary based on your actual namespace configuration.
